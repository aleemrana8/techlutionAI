# Techlution AI - End-to-End AI-Powered IT Solutions

A modern, premium, and responsive React website for Techlution AI — an IT company providing end-to-end AI solutions to minimize workforce and make life easier, with stunning UI/UX design.

## 🚀 Features

- **Modern Design**: Premium, futuristic UI with glassmorphism effects
- **Smooth Animations**: Framer Motion for fluid interactions
- **Fully Responsive**: Mobile, tablet, and desktop optimized
- **6 Service Categories**: AI & ML, Computer Vision, Automation, Healthcare AI, DevOps, Data Pipelines
- **8-Step Process**: Detailed workflow showing our development process
- **10 Projects**: Comprehensive project showcase with features
- **Contact Section**: Complete contact form and information
- **Production Ready**: Clean, modular, and scalable code structure

## 📋 Sections

1. **Hero Section** - Eye-catching introduction with CTA buttons
2. **Services Section** - Six core service offerings
3. **Process Section** - Eight-step development workflow
4. **Projects Section** - Ten detailed project examples
5. **Contact Section** - Contact form and company information
6. **Footer** - Quick links and contact details

## 🛠 Tech Stack

- **React 18** - UI framework
- **TypeScript** - Type safety
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animation library
- **Lucide React** - Icon library

## 📦 Installation

### Prerequisites
- Node.js 16+ and npm/yarn

### Setup

```bash
# Clone or navigate to the project
cd "A. Tech"

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🚀 Development

The project structure follows best practices with reusable components:

```
src/
├── components/
│   ├── common/
│   │   ├── Button.tsx      # Reusable button component
│   │   ├── Card.tsx        # Card wrapper component
│   │   └── Section.tsx     # Section wrapper component
│   ├── sections/
│   │   ├── Hero.tsx        # Hero section
│   │   ├── Services.tsx    # Services showcase
│   │   ├── Process.tsx     # Process workflow
│   │   ├── Projects.tsx    # Projects showcase
│   │   └── Contact.tsx     # Contact form
│   └── layout/
│       ├── Header.tsx      # Navigation header
│       └── Footer.tsx      # Footer with links
├── App.tsx                 # Main app component
├── main.tsx                # Entry point
└── index.css              # Global styles with Tailwind
```

## 🔧 Customization

### Colors
Edit `tailwind.config.js` to customize colors and gradients.

### Content
- Update contact information in `Contact.tsx` and `Footer.tsx`
- Modify services in `Services.tsx`
- Update projects in `Projects.tsx`
- Customize process steps in `Process.tsx`

### Animations
Adjust animation timing and effects in individual components using Framer Motion props.
